# Overview

This is the v0.18.4.5 release of the Monero software. This release fixes a bug with Ledger hardware wallet.

Some highlights of this release are:

- Ledger: fix Ledger Monero app crash (#[10234](https://github.com/monero-project/monero/pull/10234))
- Ledger: add support for Ledger Nano Gen5 (#[10243](https://github.com/monero-project/monero/pull/10243))
- Daemon: fix race condition causing dropped connections during sync (#[10257](https://github.com/monero-project/monero/pull/10257))
- Wallet: fix edge case where key images remain marked unspent (#[10255](https://github.com/monero-project/monero/pull/10255))
- Improve terminal color detection (#[10268](https://github.com/monero-project/monero/pull/10268))
- Minor bug fixes and improvements

The complete list of changes is [available on GitHub](https://github.com/monero-project/monero/compare/v0.18.4.4...v0.18.4.5), along with [the source code](https://github.com/monero-project/monero/tree/v0.18.4.5).
